json
	JSON: formato de objeto de javascript
	ej: {"prop1": 1, "prop2": null, "prop3": true, "prop_a": [1,2,3,"e"], "prop_o": {"r":1}, "prop_s": "string1"}

	tipos de datos de visual basic compatibles con los de javascript (json.vb)
		javascript                visual basic
			object {}                jsObject , Dictionary(Of String, Object), IDictionary
			array  []                jsArray , List(Of Object), IEnumerable
			numeric                  Double (el único tipo numerico de js es Double)
			string                   String
			boolean                  Boolean
			null                     Nothing
			function                 jsFunction

			
	json.parse(r As TextReader) As Object -> pasa un textReader que representa un json a datos manejables por visual basic
		Los posibles valores devueltos son:
			jsObject
			jsArray
			String
			Double
			Boolean
			Nothing
			
	json.objectToUnEval(obj As Object) As String -> pasa un datos de visual basic a un string json
		similar a la función de js JSON.stringify pero acepta funciones de javascript
		Tipos de datos de visual basic aceptables
			IDictionary -> pasara a object
			IEnumerable -> parara a array
			String
			Double, int, long, y otros tipos numericos -> pasara a numeric
			Boolean
			Nothing -> pasara a null
			jsFunction -> pasara a función de javascript(string sin comillas)
