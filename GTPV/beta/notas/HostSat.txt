- Host y Satelite

Host: applet java que crea servidor http
Sat:  petición de pagina web y comunicación con la aplicación en Ajax, datos json

Los objetos y funciones que maneja el Host estan el objeto window.H

Sat Local: 
	satelite que esta en Host, las comunicaciones son similares a un Sat normal.
	Las peticiones al Sat Local no se realizan en el mismo thread js que la petición, se hacen con
	setTimeout(f,0), así se simula que es un Sat normal y evitar que la respuesta se
	genere en el mismo thread js que la petición.
